Profile自己紹介

ポートフォリオをご覧頂きありがとうございます。WEBエンジニアとして就職活動中の 米倉 康弘（よねくら やすひろ）と申します。

1st Career経歴１

大学を一身上の都合により中退後、地元の大手小売店に就職。日配食品を担当する部門のマネージャーを約４年間務めました。

商品の発注、販売計画の立案といった業務を主として行っていたため、WEB業界とジャンルは異なりますが、マーケティングについての知見があります。

また、マネージャーとしてパート従業員のマネジメントを行っていたほか、労働組合の支部長を務めていた経験から、人の意見を汲み取り、まとめる力を身に着けられたと思っています。

2nd Career経歴２

ものづくりに関わる仕事に携わりたいとの思いから自動車部品の製造会社に転職。製品の設計を行う部署で約１年半勤務しました。

業務内容は設計日程の調整、部門間のすり合わせなどで、前職の経験を活かしながら、交渉力・調整力といったコミュニケーションスキルを高めました。

その後、家業を手伝うため2019年4月に退職。現在はキャリア形成のためWEB業界での就職を目指しプログラミングを独学しています。

Character性格

やわらかい性格で、人当たりのよさに定評があります。基本的に人を嫌いにならないので、誰とでも仲良くなれるタイプだと思っています。

食べることが大好きで、美味しいものを食べているときに一番の幸せを感じます。代償として最高90キロ代まで体重が増加しました。現在はピークから20キロのダイエットに成功し、現在も筋トレを日課としています。

Codingコーディングスキル

コーディングに関するスキルを紹介します。

HTML

  • タグの意味を理解し、W3C標準に準拠したセマンティックなマークアップができます。
  • APIを利用した、各種SNS、Googleマップ、Youtubeなどの埋め込み、シェアボタンの設置に対応出来ます。
  • 構造化マークアップを用いてリッチスニペットに対応できます。

CSS

  • サイズの相対指定、メディアクエリなどを用いてリキッドレイアウトのレスポンシブデザインに対応できます。
  • Bootstrapを用いたコーディングを行うことができます。
  • FLOCSSなどの設計思想を理解し、保守性の高いクラス設計を行うことができます。

JavaScript

  • 求める動きに必要なjQueryプラグイン、jsライブラリを選択し実装することができます。
  • JavaScript,jQueryの基本構文を理解し、簡単な動きであればゼロベースで書くことができます。
  • DOMの概念を理解し、動的にページを書き換えることができます。

WordPress

  • WordPressを用いたブログ運営経験があり、ひと通りの使用方法を理解しています。
  • 静的ページのWordPres化を行うことができます。
  • PHPの基本構文・WordPress関数を理解しており、投稿の取得・表示を操作できます。
  • ローカル開発環境の構築からサーバーへの実装までを行うことができます。

Sass

チェックリスト

コーディングの品質を担保するためチェックリストを使用しています。

チェックリストを表示する

Check List

  • 誤字脱字がないか
  • フォントが適切に設定されているか
  • 色はデザインカンプ通りか
  • デザインカンプの効果をCSSで表現できているか（box-shadow、opacityなど）
  • hover時の挙動は正しく設定されているか
  • JavaScriptの挙動は正しいか
  • デザインカンプとWebサイトを見比べてズレがないか
  • スムーズにレスポンシブされるか
  • 水平スクロールしないか
  • 改行位置は適切か
  • スマホ（320px）、タブレット（768px）、PC（カンプ）の3パターンの表示に崩れはないか
  • Chrome、FireFox、Safari、Edge（IE）で表示崩れが起きていないか
  • 担保しているブラウザverを把握しているか
  • メタ情報は正しく設定されているか
  • 見出し構造が不自然ではないか
  • 画像の形式・サイズ・圧縮率は適切か
  • altは正しく設定されているか
  • 閉じタグは正しく対応しているか
  • W3Cの標準規格に沿っているか（HTML、CSS）
  • リンク切れが起きていないか
  • コンソールエラー・ネットワークエラーが出てないか
